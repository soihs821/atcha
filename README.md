# 앗챠피디아 (Atcha Pedia)

> 왓챠 피디아 사이트의 클론 코딩입니다.


OTT 서비스 중 하나인 왓챠를 React를 이용하여 클론 코딩한 사이트입니다.

![](../header.png)

## Install

OS X & 리눅스:

```sh
yarn 
```

윈도우:

```sh
edit autoexec.bat
```
